# Mints::User::Helpers::ObjectActivities

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_object_activities(options) #=> Get a collection of object activities.

user.get_object_activity(id, options) #=> Get an object activity.

user.create_object_activity(data) #=> Create an object activity.

user.update_object_activity(id, data) #=> Update an object activity info.

user.delete_object_activity(id) #=> Delete an object activity.
```
