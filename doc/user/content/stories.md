# Mints::User::Content::Stories

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.duplicate_story(id, data) #=> Duplicate a story.

user.get_stories(options, use_post) #=> Get a collection of stories.

user.get_story(id, options) #=> Get a story info.

user.create_story(data) #=> Create a story.

user.update_story(id, data) #=> Update a story info.

user.delete_story(id) #=> Delete a story.
```
